You mean with omitted recovery? That's hardly "properly".

Cancel-unaware code is not meant to be canceled. 

It's as simply as that.

 while (true)
  try
   {
    body();
   }
  catch (...)
   {
    recover();
    handle_thread_termination_request(); /* http://tinyurl.com/8q3qt */
   }
