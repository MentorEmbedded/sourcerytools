The 2006Q1 release of the CodeSourcery GNU Toolchain for ARM Processors
is now available from:

http://www.codesourcery.com/gnu_toolchains/arm/

The 2006q1-3 release contains the following new features:

* Upgrade to GCC 4.1

The 2006Q1 release is based on GCC 4.1.0.

* Support for ARM V7, including Cortex-M3

This release supports the ARM V7 architecture, including the Cortex-M3
core. Pure Thumb-2 runtime libraries are included for execution on ARM
V7 CPUs that do not support the ARM operating mode.
