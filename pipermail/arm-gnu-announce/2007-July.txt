The 2007q1-21 release of Sourcery G++ Lite for ARM is now available from:

  http://www.codesourcery.com/gnu_toolchains/arm/

This release is an update of the 2007q1-10 release, containing fixes for
the following defects in the 2007q1-10 release:

*  The NEON MVFRO and MVFR1 registers are now supported by the assembler.

* A variety of defects in the Thumb-2 assembler have been corrected.

* EABI attribute merging has been improved.

* Diagnostics about using the host /usr directories on the target can
now be suppressed with -Wno-poison-system-directories.

There are no new features in this release.  The next major update
release will be the 2007q3 release, scheduled for October 1st, 2007.

Enjoy!
