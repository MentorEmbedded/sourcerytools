The 2005Q3-2 release of the CodeSourcery GNU Toolchain for ARM Processors
is now available from:

http://www.codesourcery.com/gnu_toolchains/arm/

This is a minor update release based on the 2005q3-1 release.

The main issues fixed are:

- armv5te Codegen bug with 64-bit types
- missing assembler flags when building linux kernel modules
- Incorrect preprocessor defines with -fsingle-precision-constant
