The 2005-Q3 release of the CodeSourcery GNU Toolchain for ARM Processors
is now available from:

http://www.codesourcery.com/gnu_toolchains/arm/

Among other improvements, this release contains support for the new
Thumb-2 architecture, which is used in the Cortex M3, and will be
present in other forthcoming ARM cores.
