CodeSourcery is pleased to announce the availability of the 2008q1
Sourcery G++ Lite Edition release for ARM processors.  This is
available for download from:

	http://www.codesourcery.com/gnu_toolchains/arm

New features in this release include:

* Optimizations for ARM Cortex-R4F and Cortex-A9 cores.

* GCC has been upgraded to GCC 4.2.3.

* A number of miscellaneous improvements ranging from profiling to 
debugging.

Enjoy!
