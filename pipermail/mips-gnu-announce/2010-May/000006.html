CodeSourcery is pleased to announce the availability of the 2010 Q1
Sourcery G++ Lite Edition release for MIPS processors.  This is
available for download from:

	http://www.codesourcery.com/gnu_toolchains/mips

Some of the new features in this release include:

* Support for the microMIPS™ ISA, a compact encoding of the MIPS 
instruction set.
* Improved software floating-point performance for bare-metal and/or 
RTOS environments.
* Many components, including GDB and EGLIBC, have been upgraded.

Refer to the Getting Started Guide (available from the download page)
for new release notes.
