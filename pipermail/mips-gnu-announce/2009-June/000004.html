The 2009q1 release of Sourcery G++ Lite for MIPS ELF is now available.

Important new features in this release include:

* Emulation of hardware floating-point instructions on bare-metal systems

* Board support package for MIPSsim

* Faster compilation times

* Loop optimization improvements

* Workarounds for MIPS 24K errata

* An upgrade to GCC 4.3.3

* Numerous bug fixes

The release can be downloaded from:

http://www.codesourcery.com/gnu_toolchains/mips/download.html
