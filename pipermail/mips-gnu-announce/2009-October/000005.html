The 2009q3 releases of Sourcery G++ Lite for MIPS ELF and MIPS 
GNU/Linx are now available.

The Fall 2009 release features:

     * Upgrade to GCC 4.4.3
     * Support for MIPS 1004K
     * Profiling support on bare-metal systems
     * Support for writing interrupt handlers in C code
     * Numerous bug fixes

The release can be downloaded from:

http://www.codesourcery.com/gnu_toolchains/mips/download.html
