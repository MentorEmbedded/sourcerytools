CodeSourcery's first release of the GNU Toolchain for ColdFire
processors is now available!

This release is based on GCC 4.1 and contains the entire GNU Toolchain:
the GNU C/C++ compilers, Binutils, GDB, and the Newlib C library.

Visit:

http://www.codesourcery.com/gnu_toolchains/coldfire

to download CodeSourcery's ColdFire toolchains.

ColdFire toolchains for uClinux and GNU/Linux will be coming soon!
