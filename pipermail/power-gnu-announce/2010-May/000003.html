CodeSourcery is pleased to announce the availability of the 2010q1
Sourcery G++ Lite Edition release for Power processors.  GNU/Linux,
EABI, and ELF versions are available for download:

http://www.codesourcery.com/gnu_toolchains/power

Some of the new features in this release include:

- Hand-optimized routines for common C and math library routines for
  E500 processors;

- Code-generation improvements for E500mc and E500 processors;

- Upgrades to GDB and GLIBC.

Please refer to the Getting Started Guide (on the download page) for
complete release notes.

Happy hacking!

-Nathan
