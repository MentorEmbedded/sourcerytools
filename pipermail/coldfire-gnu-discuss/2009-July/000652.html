Hi Thierry,

You might not need to wait for new release.
Try to modify linker file to reserve 24 bytes right after end of vector 
table (addr 0x400).
To do this I use following (for MCF52235):
    LONG (0x01234567)
    LONG (0x89abcdef)
    LONG (0xffffffff)
    LONG (0x00000000)
    LONG (0x00000000)
    LONG (0xffffffff)

Best regards,
Sergei.
