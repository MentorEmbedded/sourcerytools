CodeSourcery is pleased to announce the availability of the initial 
Sourcery G++ Lite Edition release for SuperH uClinux.  This is available 
for download from:

     http://www.codesourcery.com/sgpp/lite/superh

(This page provides both uClinux and full GNU/Linux releases.)

Key features of this new toolchain:

* Support for SH-2A uClinux.

* Support for dynamic shared libraries in a shared memory space using 
FDPIC (Function Descriptor Position Independent Code).
